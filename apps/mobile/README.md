# Dating App - Mobile (React Native)

This is the React Native mobile app for the Dating Management App.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables in `.env`:
   ```
   EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   EXPO_PUBLIC_AMPLITUDE_API_KEY=your_amplitude_key
   ```

3. Run the app:
   ```bash
   npx expo start
   ```

## Features

- Authentication (Google, Facebook, Magic Link)
- Partner management
- Partner notes
- Google Calendar sync
- Amplitude analytics
